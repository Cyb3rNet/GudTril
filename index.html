GudTril by Cyb3rNet

Another PHP GitHub API library!

Although I have found another PHP GitHub library lately this one is a bit different. It also implements all the GitHub API services and attempts to facilitate the requests to the GitHub API through a class instantiation of its category and a method call for each service request. I'm sure you'll enjoy using it and hope it will make your programming life simpler. Cheers!

Dependencies

Uses the PHP curl library.

Install

Unpack the library in your lib folder and include the "github.api.inc.php" file. You will have to edit the config file "github.confs.inc.php" and add your user and token identifiers.
